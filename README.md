# 👻 Monster Catcher

A fun arcade game built with **Kotlin** and **Jetpack Compose** for Android.

## 🎮 About

Monster Catcher is a fast-paced arcade game where players must catch the friendly ghost while avoiding dangerous characters. The game features smooth animations, random movement, a countdown timer, and a high-score system.

## ✨ Features

* 👻 Catch the friendly ghost to earn points.
* 💀 Avoid dangerous characters that reduce your score.
* 🤦‍♀️ Watch out for game-ending traps.
* ⏰ 30-second challenge mode.
* 🏆 High-score tracking.
* 🎯 Smooth animated movement.
* 📱 Built entirely with Jetpack Compose.

## 🛠️ Built With

* Kotlin
* Jetpack Compose
* Android Studio
* Coroutines
* State Management

## 🎯 How to Play

1. Press **Start Game**.
2. Tap the friendly ghost to gain points.
3. Avoid dangerous characters.
4. Score as many points as possible before the timer runs out.
5. Beat your highest score!
